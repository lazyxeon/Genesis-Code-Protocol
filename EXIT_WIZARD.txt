Exit Wizard
===========

1. Validate schema and run tests
   make install
   pre-commit run --all-files
   make test

2. Build container, generate SBOM, sign artifacts
   make build
   make sbom
   make sign

3. Launch canary deployment with OTEL enabled
   make canary

4. Run 15-minute SLO probe; promote if pass else rollback
   ./scripts/slo_probe.sh --minutes 15 || echo "rollback"

5. Emit release receipt
   commit=$(git rev-parse HEAD)
   image=$(docker images -q gha-workflow-resilience:latest)
   printf '{"commit":"%s","image":"%s"}' "$commit" "$image" > release_receipt.json
